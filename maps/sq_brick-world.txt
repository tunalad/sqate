* Construction *

Base                    : Modified `rayish_brick-world` (https://gamebanana.com/mods/93321)
Build Time              : idk, few seconds due to leaving an intentional leak
Editor(s) used          :
                          - trenchbroom (map editor)
                                https://trenchbroom.github.io/
                          - qthon (wad editor)
                                https://github.com/tunalad/qthon
                          - texmex (to convert wad3 textures to wad2)
                                https://valvedev.info/tools/texmex/
                          - winbspc (to decompile original .bsp to .map)
                                https://gamebanana.com/tools/5030
                          - bsp texture tools (to extract original map's textures (not included with this map))
                                https://gamebanana.com/tools/5900
Known Bugs              : IF NOT YET OBVIOUS, THERE'S AN INTENTIONAL LEAK LEFT IN THE MAP
May Not Run With        : with engines that have a problem with maps having leaks
Tested With             : vkquake, fteqw, darkplaces
